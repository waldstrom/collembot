# Data provenance (CC-BY-4.0)

All data in this folder is provided under **CC-BY-4.0**. The datasets are organized by source site/lab and include slide images with LabelMe JSON annotations where applicable.

## Folder contents
- `inference-examples/`
  - Example slide images for inference demonstrations and sanity checks.
  - Filenames reflect study/site metadata (e.g., Amsterdam LUFA series).
- `train-labelme-amsterdam/`
  - LabelMe annotations and slide tiles collected by the Amsterdam collaboration site.
- `train-labelme-basel/`
  - LabelMe annotations and slide tiles collected by the Basel collaboration site.
- `train-labelme-bayreuth/`
  - LabelMe annotations and slide tiles collected by the Bayreuth collaboration site.
- `train-labelme-coimbra/`
  - LabelMe annotations and slide tiles collected by the Coimbra collaboration site.
- `train-labelme-cai_pyramids/`
  - LabelMe annotations and slide tiles from the CAI pyramids collection.

If you use these datasets in publications or derivative works, please comply with CC-BY-4.0 and provide the required attribution (see the root `README.md` citation).
